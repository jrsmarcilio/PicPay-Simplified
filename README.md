# PicPay-Simplified

O PicPay Simplificado é uma plataforma de pagamentos simplificada. Nela é possível depositar e realizar transferências de dinheiro entre usuários. Projeto Realizado para o Desafio técnico do PicPay.

## Modelagem de dados

### Tipos de usuários
  - Usuário comum
    - Nome Completo
    - CPF
      - único 
    - Email
      - único
    - Senha
    - Saldo
  
  - Lojista
    - Nome Completo
    - CNPJ
      - único
    - Email
      - único
    - Senha
    - Saldo
